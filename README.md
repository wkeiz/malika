## ✨ Особенности сайта

### Минималистичный дизайн

Сайт выполнен в минималистичном стиле, поскольку перегруженные страницы сложнее воспринимаются, а внимание пользователя рассеивается между различными элементами.

### Функциональность

Основной акцент сделан на удобстве использования и интерактивности сайта:

* якорные ссылки;
* кнопки навигации;
* анимации;
* переходы между разделами.
